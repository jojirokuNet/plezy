# Project Context: Plezy

## Purpose
Plezy is a modern, cross-platform Plex client built with Flutter. It supports Android (including Android TV), iOS, macOS, Windows, and Linux.

## Tech Stack
- **Frontend:** Flutter / Dart
- **Players:** MPV (native integration on Windows/macOS/Linux), ExoPlayer (Android).
- **Architecture:** Provider for state management, clean architecture principles.

## Guidelines for Gemini CLI
- **Performance:** Focus on smooth UI transitions and efficient memory usage, especially for low-powered TV devices.
- **Security:** Ensure Plex tokens and sensitive user data are handled securely.
- **Clean Code:** Adhere to Dart linting rules and maintain high readability.
- **Tone:** Professional, direct, and collaborative.
